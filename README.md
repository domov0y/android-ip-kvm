# android-ip-kvm
попытка  сотворить  ip kvm  из  платы захвата и старого  андроид смартфона

## что нужно?
1. андроид устройство с otg
2. плата захвата из тех, что распознаются андроид устройствами
3. ардуина типа nano или uno
4. usb хаб

## как это должно работать
0. в андроид устройство заливают ip webcam
1. к андроид смартфону через  хаб подключается плата захвата. 
2. запускают эту самую программу  получают  адрес и порт где  можно увидеть трансляцию. (собственно с видео на жанном этапе все)
3. к ардуине творят простенькую макетку чтобы обеспечить работу vusb. картинки есть  на странице проекта easyhid
4. в ардуину заливают прошивка на основе библиотеки easyhid. она конвертирует команды переданные через uart в  события hid  (не знаю как правильно обозвать)
(прошивка еще не проверена.)

дальше магия которую я пока не умею. 
требуется костыль который будет пересылать полученные из браузера коды клавиш в uart. реализовать это можно либо двумя костылями 
костыль №1: программа которая будет с прослушиваемого  tcp  порта  пересылать данные  в  uart (вроде что то старое есть на  гитхабе. надо перепроверять)
костыль #2: http сервер который будет пересылать полученные  данные  в  тот самый  tcp  порт

либо через лунапарк на костылях: с созданием своей  программы на базе библиотеки PhysicaloidLibrary и простенького http сервера
